Documents index

1        Starting point

·         MTS_quick_start.doc: => the quick start document for the first use of the product. It contains installation procedure, application starting and documentation content.

2        Presentation document

·         MTS_pres_technical.ppt: => a commercial presentation to idea of the product benefit for the customer: (Powerpoint application required)

·         MTS_pres_demos.ppt: => a demos presentation to show by examples the main features of the product: It illustrates the main use cases of the product: functional, load, server simulation testing use cases for SIP, Diameter, RTP and HTTP protocols (Powerpoint application required)

3        User manuals

User manuals allow users to have a complete description of the MTS features. They are divided by modules and you have one of them per module.

 

Here is the complete list of the user manuals :

 

General features :

·         MTS_user_manual_core.doc: => Generic features for test/testcase/scenario running, input/output files processing, protocols management and some operations independent of protocols.

 

·         MTS_user_manual_gui.doc: => Graphical user interface (scripts writing, tuning debugging).

·         MTS_user_manual_cmd.doc: => Command line interface (test automation).

 

·         MTS_user_manual_master.doc: => Master / slave feature to run tests remotely in a distributed environment (scalabilty).

 

·         MTS_user_manual_genscript.doc: => Automatic scripts generation from pcap network capture (based on capture features).

MTS_user_manual_importsipp.doc: => Automatic scripts generation from sip existing scripts

 

IMS/EPS protocols :

·         MTS_user_manual_sip.doc: => SIP protocol (call setup)

·         MTS_user_manual_diameter.doc: => DIAMETER protocol (AAA features)

·         MTS_user_manual_rtp.doc: => RTP protocol (packet mode)

·         MTS_user_manual_rtpflow.doc: => RTP protocol (flow mode)

·         MTS_user_manual_http.doc: => HTTP protocol (WEB protocol incuding SOAP WEB services)

·         MTS_user_manual_gtp.doc: => GTP protocol (for EPS architecture) with V1, V2, Prime version

 

Application protocols :

·         MTS_user_manual_radius.doc: => RADIUS protocol. (old AAA features)

·         MTS_user_manual_rtsp.doc: => RTSP protocol. (IPTV session setup)

·         MTS_user_manual_smtp.doc: => SMTP protocol (email sending)

·         MTS_user_manual_imap.doc: => IMAP protocol (email query)

·         MTS_user_manual_pop.doc: => POP protocol (email query)

·         MTS_user_manual_smpp.doc : => SMPP protocol (SMS management)

·         MTS_user_manual_ucp.doc : => UCP protocol (SMS management)

·         MTS_user_manual_sigtran.doc: => SIGTRAN protocol (SS7 over IP)
=> just M3UA, ISUP, BICC, SCCP, IUA, V5UA layers and not the ASN1 encoding like TCAP, MAP, CAP, )

·         MTS_user_manual_h248.doc: => H248 protocol (megaco)
=> text encoding/ decoding supported and not binary

·         MTS_user_manual_mgcp.doc : => MGCP protocol (Media gateway control protocol)

·         MTS_user_manual_msrp.doc : => MSRP protocol (instant messaging)

·         MTS_user_manual_snmp.doc : => SNMP protocol (admin & maintenance)
=> just V1, 2 supported and not V3 for the moment

·         MTS_user_manual_pstn.doc : => PSTN protocol (old telephony) with the use of Asterisk product as a gateway (just analog line supported and ISDN line supported for the moment)

·         MTS_user_manual_stun.doc : => STUN protocol (NAT transversal)
=> just UDP transport supported and messageIntegrity (authentification procedure) not tested

·         MTS_user_manual_ethernet.doc : => ETHERNET protocol (capture only) using the wireshark pcap library

Basic transport protocols :

·         MTS_user_manual_udp.doc : => UDP protocol

·         MTS_user_manual_tcp.doc : => TCP protocol

·         MTS_user_manual_sctp.doc : => SCTP protocol

·         MTS_user_manual_tls.doc : => TLS protocol
